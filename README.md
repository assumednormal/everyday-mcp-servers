# Everyday MCP Servers

A collection of Model Context Protocol (MCP) servers for everyday life tasks.

## Overview

This repository houses custom MCP servers that help automate and streamline common daily activities.

## Servers

### Planned
- **HEB Shopping List** - Manage shopping lists for HEB grocery stores

## Structure

Each MCP server is organized in its own directory under `/servers`:

```
servers/
├── heb-shopping-list/
└── [future servers]/
```

## Development

Each server is a standalone MCP implementation that can be used with Claude Desktop or other MCP clients.

## Getting Started

Instructions for setting up and using individual MCP servers can be found in their respective directories.
